创建 JSON 帮助生成器模块,实现 `generate_json_help()` 函数。

实现要点(遵循 CONTEXT.md 的锁定决策):

1. **命令映射格式**: 返回字典而非列表
   ```python
   {
     "download": {...},
     "auth": {...}
   }
   ```

2. **命令元数据结构**: 每个命令对象包含:
   - `name`: 命令名称(字符串)
   - `description`: 命令描述(字符串,从 cmd.help 提取第一行)
   - `parameters`: 参数列表(数组)

3. **参数对象**: 每个参数包含完整字段:
   - `name`: 参数名称
   - `type`: 基础类型字符串 ("string", "integer", "boolean", "array")
   - `required`: 布尔值,标记是否必需
   - `description`: 参数帮助文本
   - `default_value`: 默认值(必需参数为 None)

4. **全局选项合并**: 每个命令的 parameters 列表必须包含全局选项:
   - `verbose` (boolean, 默认 False)
   - `quiet` (boolean, 默认 False)
   - `json_output` (boolean, 默认 False)
   - `json_help` (boolean, 默认 False)

5. **Click 元数据提取**: 使用 `ctx.to_info_dict()` 方法
   - 遍历 `cli_group.commands` 字典
   - 为每个命令创建临时上下文: `cmd.make_context(cmd_name, [], parent=None)`
   - 从 `ctx.to_info_dict()` 提取参数信息

6. **类型映射**: 将 Click 参数类型映射为简单类型:
   - `click.types.IntParamType` → "integer"
   - `click.types.BoolParamType` → "boolean"
   - `click.types.FloatParamType` → "number"
   - `click.types.StringParamType`, `click.types.Path` → "string"
   - `click.types.Tuple` → "array"
   - 其他 → "string"(默认)

7. **中文描述**: 所有描述使用中文,与项目语言一致

关键实现代码参考 RESEARCH.md Pattern 2 和 Code Examples 部分。

注意事项:
- 跳过隐藏命令(`hidden=True`)
- 处理参数 `default` 为 `click.core.UNSET` 的情况(转换为 None)
- 使用 `getattr(param, 'required', False)` 安全访问 required 属性

<task type="auto">
  <name>Task 2: 添加 --json-help 全局参数</name>
  <files>src/gallery_dl_auo/cli/main.py</files>
  <action>
修改主命令组,添加 --json-help 全局参数。

实现要点:

1. **参数定义**: 在 `@click.group()` 装饰器下添加:
   ```python
   @click.option(
       "--json-help",
       is_flag=True,
       is_eager=True,
       expose_value=False,
       callback=_handle_json_help,
       help="输出结构化 JSON 帮助信息"
   )
   ```

2. **回调函数**: 实现 `_handle_json_help(ctx, param, value)`:
   - 当 value=True 时,调用 `generate_json_help(cli)`
   - 输出 JSON 到 stdout:`print(json.dumps(help_data, ensure_ascii=False, indent=2))`
   - 调用 `ctx.exit(0)` 退出程序

3. **is_eager=True**: 确保参数在命令验证前执行(类似 --help)

4. **expose_value=False**: 不将参数传递给子命令

5. **导入语句**: 在文件顶部添加:
   ```python
   import json
   from gallery_dl_auo.cli.json_help import generate_json_help
   ```

注意事项:
- --json-help 必须在任何子命令前工作(如 `pixiv-downloader --json-help`)
- JSON 输出到 stdout,确保 `ensure_ascii=False` 以正确显示中文
- 不要修改现有的 --verbose 参数逻辑
