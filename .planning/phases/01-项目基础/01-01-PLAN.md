创建统一的日志配置模块,使用 Rich 提供彩色、格式化的输出。

实现 `setup_logging(log_level: str = "INFO") -> Console` 函数:

```python
import logging
from rich.console import Console
from rich.logging import RichHandler

def setup_logging(log_level: str = "INFO") -> Console:
    """配置 Rich 日志系统

    Args:
        log_level: 日志级别 (DEBUG, INFO, WARNING, ERROR, CRITICAL)

    Returns:
        Console: Rich Console 实例,用于后续输出
    """
    console = Console()

    # 配置 RichHandler
    handler = RichHandler(
        console=console,
        show_time=True,        # 显示时间戳
        show_level=True,       # 显示日志级别
        show_path=False,       # 不显示文件路径(避免输出过长)
        markup=True,           # 支持 Rich markup
        rich_tracebacks=True,  # 美化 traceback
        tracebacks_show_locals=True,  # 显示局部变量
    )

    # 配置 logger
    logger = logging.getLogger("gallery_dl_auo")
    logger.addHandler(handler)
    logger.setLevel(getattr(logging, log_level.upper()))

    return console
```

**关键点**:
- 返回 Console 实例供 CLI 命令使用
- show_path=False 避免输出过长
- 支持 Rich markup(如 [bold blue]...)
- 使用 "gallery_dl_auo" 作为 logger 名称

**避免**:
- 不要使用 print()(统一用 Rich Console)
- 不要创建多个 Console 实例(避免输出冲突)
