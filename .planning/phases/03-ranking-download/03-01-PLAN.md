创建 `PixivClient` 类,封装 pixivpy3 的 AppPixivAPI,提供以下功能:

1. **初始化和认证**:
   - `__init__(self, refresh_token: str)`: 使用 refresh token 初始化 API 客户端
   - 调用 `api.auth(refresh_token=refresh_token)` 进行认证
   - 处理认证失败 (网络错误、无效 token 等)

2. **排行榜获取**:
   - `get_ranking(self, mode: str = "day", date: str | None = None) -> list[dict]`: 获取排行榜数据
   - 使用 `api.illust_ranking(mode=mode, date=date)` 获取数据
   - 遍历 `json_result.illusts` 并提取关键字段:
     - id: 作品 ID
     - title: 标题
     - user.name: 作者名
     - image_urls.large: 图片 URL
   - 返回作品列表 (字典格式)

3. **分页支持**:
   - 如果 `json_result.next_url` 存在,使用 `api.parse_qs(next_url)` 获取下一页
   - 循环直到所有数据获取完毕

4. **错误处理**:
   - 定义 `PixivAPIError` 异常类 (继承 Exception)
   - 捕获 pixivpy3 的异常并转换为自定义异常
   - 提供清晰的错误消息 (认证失败、网络错误、API 限制等)

实现要点 (基于 RESEARCH.md):
- 使用 `from pixivpy3 import AppPixivAPI`

- illust_ranking 返回的 illusts 是列表,每个元素有属性访问
- 遵循现有代码风格: 使用 logging 模块记录日志
