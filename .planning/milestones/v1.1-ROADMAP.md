# Roadmap: Pixiv 排行榜下载器 (gallery-dl-auto)

## Overview

本项目通过多个阶段,从项目基础搭建到完整的 CLI 工具,最终交付一个能够自动化管理 Pixiv refresh token 并下载排行榜内容的工具。核心差异化价值在于自动化 token 管理——用户首次手动登录后,所有 token 刷新和过期处理都自动完成,无需手动从浏览器开发者工具复制。每个阶段都交付一个可验证的完整能力,逐步构建从认证、下载、输出到用户体验的完整功能。
