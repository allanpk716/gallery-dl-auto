# Roadmap: Pixiv 排行榜下载器 (gallery-dl-auto)

## Overview

本项目通过 8 个阶段,从项目基础搭建到完整的 CLI 工具,最终交付一个能够自动化管理 pixiv refresh token 并下载排行榜内容的工具。核心差异化价值在于自动化 token 管理——用户首次手动登录后,所有 token 刷新和过期处理都自动完成,无需手动从浏览器开发者工具复制。每个阶段都交付一个可验证的完整能力,逐步构建从认证、下载、输出到用户体验的完整功能。

## Phases

- [ ] **Phase 1: 项目基础** - 搭建项目结构、配置管理、核心依赖
- [ ] **Phase 2: Token 自动化** - 实现 refresh token 自动获取、存储和刷新
- [ ] **Phase 3: 排行榜基础下载** - 实现基础排行榜下载和图片获取
- [x] **Phase 4: 内容与元数据** - 完整的元数据获取和路径配置
- [x] **Phase 5: JSON 输出** - 结构化 JSON 输出和错误信息
- [ ] **Phase 6: 多排行榜支持** - 扩展支持周榜、月榜等多种排行榜
- [ ] **Phase 7: 错误处理与健壮性** - 完善的错误处理和增量下载
- [ ] **Phase 8: 用户体验优化** - 进度显示、速率控制和 CLI 完善
